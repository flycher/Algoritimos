
# Algoritimos

## *Estrutura de Dados:*

* Ordenação
  * Bubble Sort
    * Ω(n) | θ(n²) | O(n²)
  * Insertion Sort
    * Ω(n) | θ(n²) | O(n²)
  * Merge Sort
    * Ω(n log(n)) |  θ(n log(n)) | O(n log(n))
  * Heap Sort
    * Ω(n log(n)) | θ(n log(n)) | O(n log(n))
  * Quick Sort
    * Ω(n log(n)) | θ(n log(n)) | O(n²)
  * Counting Sort
    * Ω(n) | θ(n) | O(n)
* Pilha
  * LIFO
    * Topo: O(1) | Inserir: O(1) | Remover: O(1)
* Fila
  * FIFO
    * Cabeça: O(1) | Inserir: O(1) | Remover: O(1)
* Lista
  * Guarda valores, a ordem dos valores não importa
    * Pegar elemento: O(n) | Inserir: O(1) | Remover: O(n)
